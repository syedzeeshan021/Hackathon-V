## Product Summary

TechCorp provides an API management platform that helps companies:
- Build and deploy APIs faster
- Monitor API performance and usage
- Secure APIs with authentication and rate limiting
- Document APIs automatically
- Analyze API traffic and trends

### Key Features

1. **API Gateway** - Route, transform, and secure API traffic
2. **Developer Portal** - Auto-generated API documentation
3. **Analytics Dashboard** - Real-time API metrics and insights
4. **Authentication** - OAuth2, JWT, API keys out of the box
5. **Rate Limiting** - Protect APIs from abuse
6. **Monitoring** - Alerts and notifications for API issues

## Brand Voice

**Tone:** Helpful, professional, technically competent

**Style Guidelines:**
- Use clear, concise language
- Avoid jargon when possible
- Be empathetic to customer frustration
- Always offer next steps
- Acknowledge issues before solving them

**Example Responses:**

❌ Too robotic: "Your password has been reset. Check email."

✅ Better: "I've reset your password for you! You should receive an email shortly with instructions to set a new password. Let me know if you don't see it within a few minutes."
